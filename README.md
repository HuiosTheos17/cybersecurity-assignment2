# Cybersecurity Assignment2

## Overview

This repository contains the results of two cybersecurity tasks:

1. Performing a vulnerability scan on a demo website.
2. Analyzing a phishing attack scenario and identifying indicators of compromise.

The goal of this assignment is to demonstrate basic cybersecurity analysis skills including vulnerability identification and phishing detection.

---

## Tools Used

- OWASP ZAP (Zed Attack Proxy)
- Online vulnerability scanners
- GitHub for documentation

---
## Repository structure 
---

## Task 1: Vulnerability Scanning

A vulnerability scan was performed on a demo web application designed for security testing.

Target Website:
http://testphp.vulnweb.com

The scan identifies possible security weaknesses such as:

- Cross-Site Scripting (XSS)
- SQL Injection
- Missing Security Headers

The full report can be found in:

vulnerability-scan/scan-report.md

---

## Task 2: Phishing Attack Analysis

A simulated phishing email was analyzed to determine signs of malicious activity.

Indicators such as:

- Suspicious sender addresses
- Fake domains
- Urgent language
- Credential harvesting attempts

were identified.

The full report can be found in:

phishing-analysis/phishing-report.md

---

## Conclusion

This assignment demonstrates how vulnerability scanning tools can identify security weaknesses in web applications and how phishing attacks can be recognized through careful inspection of email and web links.
